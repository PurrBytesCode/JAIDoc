# DocumentationService — JDK Source → JSON Documentation Pipeline (Approach A)

## Context

The `DocumentationService` is currently an empty `@Service` stub. The goal is to implement a pipeline that:
1. Downloads a JDK source archive (JDK version must be specified as parameter)
2. Extracts the source zip to a working directory
3. Runs `javadoc` with the custom `JsonDoclet` to produce JSON documentation
4. Stores output organized by JDK version — one directory per version, persisting across runs

The execution model is **async with progress callback** (returns `CompletableFuture<Path>`, accepts `Consumer<Double>`). The work directory is **configurable** via `doclet.work.directory`.

**Approach A — Fat JAR on Classpath**: Resolve the Spring Boot fat JAR path at runtime and pass it directly as `-docletpath` to `javadoc`. Simplest and most direct.

## Approach A — Fat JAR on Classpath

### Overview

Resolve the Spring Boot fat JAR path at runtime and pass it directly as `-docletpath` to `javadoc`. The doclet class lives inside the fat JAR alongside all other application classes.

### JAR Path Resolution

Three strategies, tried in order:

1. **ProtectionDomain** — `DocumentationService.class.getProtectionDomain().getCodeSource().getLocation()`. In production (`java -jar`), this returns the fat JAR URI. Validate it's a Spring Boot fat JAR by checking for `BOOT-INF/classes` entry inside the ZIP.

2. **Classpath scan** — In dev/IDE mode, `System.getProperty("java.class.path")` contains multiple entries. Look for a `JAIDoc-*.jar` among them.

3. **Fallback: null** — If no JAR found (dev mode, no built JAR), pass no `-docletpath`. The `javadoc` tool will use its own classpath — this works in dev if the doclet classes are on the same JVM classpath.

### Pipeline Flow

```java
public CompletableFuture<Path> generateJdkDocumentation(String version, Consumer<Double> progressCallback) {
    // Phase weights: download 40%, extract 15%, javadoc 45%
    final double DOWNLOAD_WEIGHT = 0.40;
    final double EXTRACT_WEIGHT = 0.15;
    final double JAVADOC_WEIGHT = 0.45;

    Consumer<Double> downloadProgress = p -> {
        if (progressCallback != null) progressCallback.accept(p * DOWNLOAD_WEIGHT);
    };

    return jdkSourceDownloader.downloadSource(version, downloadProgress)
        .thenApply(zippedPath -> extractSourceZip(zippedPath, version,
            p -> progressCallback.accept(DOWNLOAD_WEIGHT + p * EXTRACT_WEIGHT)))
        .thenApply(sourceRoot -> runJavadocDoclet(sourceRoot, version,
            p -> progressCallback.accept(DOWNLOAD_WEIGHT + EXTRACT_WEIGHT + p * JAVADOC_WEIGHT)))
        .exceptionally(ex -> {
            log.error("JDK documentation generation failed for version {}: {}", version, ex.getMessage());
            throw new CompletionException(ex);
        });
}
```

### Zip Extraction

```java
private Path extractSourceZip(Path zipFile, String version, Consumer<Double> progressCallback) throws IOException {
    Path extractDir = workDirectory.resolve("jdk-sources").resolve(version);

    // Idempotent: skip if already extracted
    Path existingRoot = findSourceRoot(extractDir);
    if (Files.exists(existingRoot)) {
        log.info("Source already extracted at {}", existingRoot);
        return existingRoot;
    }

    Files.createDirectories(extractDir);
    try (ZipInputStream zis = new ZipInputStream(Files.newInputStream(zipFile))) {
        ZipEntry entry;
        while ((entry = zis.getNextEntry()) != null) {
            Path dest = extractDir.resolve(entry.getName()).normalize();
            // Zip-slip protection
            if (!dest.startsWith(extractDir)) {
                log.warn("Skipping zip-slip entry: {}", entry.getName());
                zis.closeEntry();
                continue;
            }
            if (entry.isDirectory()) {
                Files.createDirectories(dest);
            } else {
                Files.createDirectories(dest.getParent());
                Files.copy(zis, dest, StandardCopyOption.REPLACE_EXISTING);
            }
            zis.closeEntry();
        }
    }

    Path sourceRoot = findSourceRoot(extractDir);
    log.info("JDK source extracted to: {}", sourceRoot);
    return sourceRoot;
}

private Path findSourceRoot(Path extractDir) throws IOException {
    // The zip contains a single top-level directory (e.g. "jdk-25.0.3-ga/")
    try (DirectoryStream<Path> stream = Files.newDirectoryStream(extractDir)) {
        Iterator<Path> it = stream.iterator();
        if (it.hasNext()) {
            Path entry = it.next();
            if (Files.isDirectory(entry)) return entry;
        }
    }
    return extractDir;
}
```

### Javadoc ProcessBuilder

```java
private Path runJavadocDoclet(Path sourceRoot, String version, Consumer<Double> progressCallback) {
    // 1. Create temp output dir for javadoc (cleaned after copy)
    Path tempOutputDir = workDirectory.resolve("javadoc-out").resolve(version);
    Files.createDirectories(tempOutputDir);

    // 2. Resolve javadoc binary
    String javaHome = System.getProperty("java.home");
    String osName = System.getProperty("os.name").toLowerCase();
    Path javadocBin = Path.of(javaHome, "bin", osName.contains("win") ? "javadoc.exe" : "javadoc");

    if (!Files.exists(javadocBin)) {
        throw new IllegalStateException("javadoc binary not found at: " + javadocBin);
    }

    // 3. Resolve fat JAR path
    String fatJarPath = resolveFatJarPath();

    // 4. Build command
    List<String> command = new ArrayList<>();
    command.add(javadocBin.toString());

    if (fatJarPath != null) {
        command.add("-docletpath");
        command.add(fatJarPath);
    }

    command.add("-doclet").add("com.purrbyte.ai.doclet.JsonDoclet");
    command.add("-sourcepath").add(sourceRoot.toString());
    command.add("-d").add(tempOutputDir.toString());
    command.add("--pretty");
    command.add("-subpackages").add("java;jdk;com.sun;javax;org.ietf;org.w3c;org.xml.sax");

    log.info("Executing javadoc: {}", command);

    ProcessBuilder pb = new ProcessBuilder(command);
    pb.redirectErrorStream(true);

    Process process = pb.start();

    // Stream output and track progress
    CompletableFuture<Void> progressTicker = startProgressTicker(progressCallback);

    try (BufferedReader reader = new BufferedReader(
            new InputStreamReader(process.getInputStream(), StandardCharsets.UTF_8))) {
        String line;
        while ((line = reader.readLine()) != null) {
            log.debug("[javadoc] {}", line);
        }
    }

    int exitCode = process.waitFor(JAVADOC_TIMEOUT_SECONDS, TimeUnit.SECONDS);
    progressTicker.cancel(true);

    if (exitCode != 0) {
        cleanupOutput(tempOutputDir);
        throw new IOException("javadoc exited with code " + exitCode);
    }

    if (!Files.exists(tempOutputDir.resolve("index.json"))) {
        cleanupOutput(tempOutputDir);
        throw new IOException("javadoc completed but index.json was not created");
    }

    // 5. Copy result to data/<version>/ (persistent backup)
    Path dataDir = outputDirectory.resolve(version);
    Files.createDirectories(dataDir);
    copyDirectory(tempOutputDir, dataDir);

    // 6. Clean up temp dir
    deleteDirectory(tempOutputDir);

    log.info("JDK documentation generated at {}", dataDir);
    return dataDir;
}
```
