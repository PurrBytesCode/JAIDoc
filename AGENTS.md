# AGENTS.md

Guidelines for AI agents working in this repository.

## Language Rules

- **Code is always written in English.** All identifiers (class names, method names, variable names, function names, constants, package names, etc.), as well as inline comments and commit messages, must be in English.
- **Documentation is always written in English.** All documentation, including Javadoc/KDoc, README files, design docs, and any other written documentation, must be in English.
